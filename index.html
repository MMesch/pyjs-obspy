About This Demo

This demo showcases Obspy running in your browser via Pyjs (Python compiled to WebAssembly). Technologies: Obspy, Pyjs, Emscripten, FDSN Web Services How it works: 1. Python environment loads in WebAssembly on page start 2. Obspy and dependencies are available in-browser 3. Data is fetched from FDSN web services 4. All processing happens locally — no server required
